# Linux Assignment Instructions

## Instructions
- Use the provided IP address, username, and password to log in to the Ubuntu virtual server  
- Perform all tasks using the terminal (CLI)  
- Use your name as part of folder naming (e.g., `hmwenda_data_engineering`)  
- Document your work using screenshots or recorded commands

## Submission Requirements
- Submit screenshots showing:
  - Successful login  
  - Folder structure  
  - File operations  
  - Permissions (`ls -l`)  
  - Command history  
- Submit as a PDF document  

**File name:** `YourName_Linux_Assignment.pdf`  

---

## Part 1: Login and Verification
Log into the Ubuntu server using the provided credentials.  

Once logged in:
- Confirm that you are using the correct account by checking your username  
- Verify your current working directory to understand where you are operating from within the system  

---

## Part 2: Folder and File Creation
- Create a main directory in your home folder using your name  
- Navigate into this directory  
- Create subfolders:
  - `raw_data`
  - `processed_data`
  - `logs`
  - `scripts`  

Within each folder, create appropriate files:
- A CSV file for raw data  
- A cleaned data file  
- A log file  
- A script file  

---

## Part 3: File Operations
- Copy one of the files to create a backup  
- Move a file from one directory to another to simulate data flow  
- Rename one of the files to reflect versioning  
- Delete any unnecessary or duplicate files  

---

## Part 4: Permissions
- Modify permissions of your main directory to restrict access to only you  
- Update file permissions so sensitive files can only be read and written by the owner  
- Make your script file executable  
- Verify all permission changes using appropriate commands  

---

## Part 5: Navigation and History
- Navigate between directories to demonstrate your understanding of the Linux file system  
- List all files, including hidden ones, to inspect directory contents  
- Access and review your command history  
- Demonstrate how to re-run a previously executed command
